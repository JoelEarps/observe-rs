# Rust Prometheus Generator

## How to run

`cargo build`
`cargo run`

## How to test

Unit Tests

`cargo test`

### Testing Strategy

Unit Tests: Coverage
Integration Tests: Running the application and hitting the endpoints to prove it works - gracefully exiting
